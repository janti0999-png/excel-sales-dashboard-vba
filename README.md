# Excel Sales Dashboard with VBA Automation

## Overview
This project is an Excel-based sales dashboard designed to track performance of sales executives and automate reporting using VBA.

## Tools Used
- Microsoft Excel
- VBA (Visual Basic for Applications)

## Key Features
- Automated report generation using VBA macros
- Interactive dashboard with charts and KPIs
- Performance tracking for sales executives
- Reduced manual effort through automation

## Key Metrics
- Total Sales
- Sales by Executive
- Target vs Achievement
- Performance Trends

## Key Insights
- Identified top-performing executives
- Highlighted gaps between targets and actual performance
- Improved reporting efficiency using automation

## Files Included
- Excel Dashboard (.xlsm)
- Dataset
- Dashboard screenshot
